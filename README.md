# anyone-can-cook.com

The personal site of Damian Pickett — product marketer and maker.
Home, selected work, writing, about, and a web CV.

Built with Astro 6, deployed on Vercel, managed through Claude Code.
See [GUIDE.md](GUIDE.md) for how everything works and how to make changes.

## Commands

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | Install dependencies                        |
| `npm run dev`     | Local dev server at `localhost:4321`        |
| `npm run build`   | Build the production site to `./dist/`      |
| `npm run preview` | Preview the production build locally        |
